Projects

Option Trading Simulator

A trading strategy simulator which can ingest historical data and evaluate and display the outcomes. This application, while still early in development will provide an invaluable analysis tool for the customer, as well as an educational resource for their customers. The application is currently built in Java and JavaFX for the prototyping stage, with plans to move into a web interface if requested.

Board Game Simulation

One of my most recent attempts to break into game design is with Godot. I decided to pick up this open source tool to have the convenience of learning design with an existing engine behind my development as well as the flexibility to practice coding in multiple languages. I will be prototyping a board game design I've been experimenting with as an online multiplayer experience, so I have a convenient and quick way to test and iterate on my designs.
